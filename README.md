## üìä What is a Pentagon Chart?

Pentagon charts visualize pairwise statistical comparisons between five conditions. They're particularly useful for:

- Displaying multiple pairwise comparisons simultaneously
- Showing which condition pairs have significant differences
- Comparing mean values across conditions
- Academic research in psychology, education, and social sciences

### Visual Elements

- **Circles**: Sized proportionally to mean values
- **Solid Lines**: Indicate statistically significant differences (with z-scores)
- **Dotted Lines**: Show non-significant differences
- **Center Label**: Names the variable being measured

## üìñ How to Use

### Creating Your First Chart

1. **Edit Data**: Click "Edit Data" button
2. **Chart Title**: Give your chart a descriptive name
3. **Center Label**: Enter the variable name (e.g., "Accuracy", "Trust")
4. **Conditions**: Enter 5 condition names and their mean values
5. **Comparisons**: Set z-scores and mark significant differences
6. **View**: Click "View Chart" to see your visualization

### Managing Multiple Charts

- **New Chart**: Create a blank chart from template
- **Duplicate**: Clone current chart for variations
- **Delete**: Remove unwanted charts
- **Dropdown**: Switch between charts instantly
